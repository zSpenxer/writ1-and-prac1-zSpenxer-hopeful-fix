**```.scene``` File**

A text-based file format based on XML, used by the Ogre3D engine. While this has the advantage of being human readable it's not especially easy to edit or to debug. 

Subfunction

A specialise but smaller function that is defined within the same file as a main function, this will help to improve modularity and code organisation
